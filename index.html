📖 Apa itu Konstanta Pendinginan (k)?

Konstanta k adalah angka yang menunjukkan seberapa cepat suatu benda berubah suhunya. Semakin besar nilai k, semakin cepat perubahan suhu terjadi.


🧪 Cara Menghitung k dari Data

Jika Anda mengetahui suhu awal dan suhu setelah waktu t, Anda bisa menghitung k dengan rumus:

k = -ln((T_t - T_ling) / (T_awal - T_ling)) / t

📐 Rumus Deret Taylor yang Digunakan

Program ini menggunakan deret Taylor untuk menghitung e^x:

e^x = 1 + x + x²/2! + x³/3! + x⁴/4! + ...

Dengan x = -k × t, program menghitung nilai eksponensial secara akurat.
